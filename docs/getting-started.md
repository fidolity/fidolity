# Getting Started with Fidolity

Welcome to Fidolity! This guide will help you get started with the platform.

## Prerequisites

Before you begin, make sure you have:

1. **A Solana Wallet**
   - Phantom (recommended)
   - Solflare
   - Or any Solana-compatible wallet

2. **Some SOL**
   - For transaction fees
   - Minimum: 0.01 SOL

3. **FDLT Tokens** (optional)
   - For staking and accessing premium features

## Step 1: Connect Your Wallet

1. Visit [fidolity.com](https://fidolity.com)
2. Click **"Connect Wallet"** in the top right
3. Select your wallet provider
4. Approve the connection request

## Step 2: Browse the Marketplace

1. Navigate to the **Marketplace**
2. Browse available AI agents
3. Filter by:
   - Category
   - Price
   - Rating
   - Executions

## Step 3: Purchase an AI Agent

1. Click on an agent to view details
2. Review:
   - Description
   - Pricing
   - Reviews
   - Demo (if available)
3. Click **"Purchase"**
4. Approve the transaction in your wallet
5. Wait for confirmation

## Step 4: Use Your Agent

1. Go to **User Panel**
2. View your purchased agents
3. Configure agent settings
4. Start using the agent!

## Step 5: Stake FDLT (Optional)

1. Navigate to **Staking**
2. Connect your wallet
3. Enter amount to stake
4. Approve transaction
5. Start earning rewards!
