# Fidolity Frontend

React + TypeScript + Vite application for the Fidolity AI Agent Marketplace.

## Tech Stack

- React 18.3
- TypeScript 5.9
- Vite 5.4
- Tailwind CSS 3.4
- React Router 7
- Solana Web3.js

## Development

```bash
# Install dependencies
yarn install

# Run dev server
yarn dev

# Build for production
yarn build

# Preview production build
yarn preview
```

## Environment Variables

Create a `.env` file:

```env
VITE_HELIUS_RPC_URL=your_helius_rpc_url
```

## Building

```bash
yarn build
```

Output will be in `dist/` directory.

## Deployment

Deploy the `dist/` folder to any static hosting:
- Vercel
- Netlify  
- GitHub Pages
- Cloudflare Pages
